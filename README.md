#  SQL Mini Project: Flipkart Mobile Sales Analysis

##  Overview
This project focuses on analyzing Flipkart mobile sales data using SQL.  
The goal is to understand pricing trends, brand performance, and customer ratings.

---

##  Objective
To analyze mobile sales data and extract meaningful insights using SQL queries.

---

##  Dataset
- Flipkart Mobile Sales Dataset  
- Total Records: 430  
- Columns: Brand, Category, Processor, RAM, ROM, Battery Capacity, Ratings, Sales Price, etc.
- Source:kaggle

---

##  Tools Used
- MySQL Workbench  
- SQL  

---

## Data Cleaning
- Checked for NULL values  
- Removed duplicates  
- Standardized data using TRIM and UPPER  

---

## Data Analysis
- Brand-wise model count  
- Average price per brand  
- Top-rated mobiles  
- Price segmentation (Budget, Mid-range, Premium)  

---

##  SQL Concepts Used
- GROUP BY, ORDER BY  
- CASE WHEN  
- JOIN (INNER, LEFT)  
- Subqueries  
- Window Functions (ROW_NUMBER, RANK, DENSE_RANK)  
- PARTITION BY  
- CTE (WITH clause)  
- Views & Index  

---

##  Key Insights
- Apple has the highest average rating  
- Budget mobiles dominate the dataset  
- High price does not always mean high ratings  
- Chinese brands have more models  

---

##  Conclusion
This project helped me gain hands-on experience in SQL and understand how to analyze real-world datasets.
